ngvc data highway

 

 

ABOUT THIS PROJECT

NGVC Data Highway is a tool that a coworker and I decided to build in our spare time to optimize a process that we perceived as suboptimal. It is a file translation, rule validation, and deployment process that translates data from Excel to XML and utilizes an IBM Sterling Secure Proxy to delegate files to various endpoints for internal consumption and displaying on live sites.

Disclaimer: this tool was never used by GM and all fragile information has been removed from the code base.
