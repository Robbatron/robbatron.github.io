drag relative path

ABOUT THE PROJECT

Drag Relative Path is an Atom extension that currently has over 650 users and 7 stars on atom.io! I desired the functionality to drag files from the tree-view (or from outside the editor) to create HTML tags with automatic relative paths, so I went ahead and coded it in CoffeeScript then published it! Atom does not have an extensive API for tree-view, so I had to get really creative! It also includes functionality to drag to other than HTML files - only then will it copy the path instead of creating an HTML tag.

I will be porting this to VSCode soon so keep an eye out!
